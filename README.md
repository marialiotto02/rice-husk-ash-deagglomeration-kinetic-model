# rice-husk-ash-deagglomeration-kinetic-model
Kinetic modeling of rice husk ash deagglomeration by ultrasound and ball milling.

This repository contains the Python code used in the study:

**"Mechanistic and kinetic comparison of rice husk ash deagglomeration by ultrasound and ball milling"**

## Description

The script implements a population balance model based on size classes to describe the redistribution of particle fractions during deagglomeration.

Two experimental datasets are analyzed:

- Ultrasound treatment
- Ball milling treatment

The model parameters are estimated using nonlinear least squares.

## Requirements

Python packages required:

- numpy
- pandas
- matplotlib
- scipy
